**ETL-Projects**
**Overview:**
End-to-end ETL pipeline using Azure Data Factory, Databricks, and ADLS.
Bronze → raw data
Silver → cleaned/transformed data
Gold → final dataset

**Setup:**
Clone repo under Databricks /Repos/sanfan0082@gmail.com/ETL-Projects.
Ensure ADF notebook activity points to the correct notebook path.
Run pipeline to process data.

**Git Workflow:**
Pull latest changes before edits.
Make changes → Save → Commit → Push.
Only push to main branch for now.
